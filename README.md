# sp500
Playing with S&amp;P 500 data, from a dataset found on Kaggle: https://www.kaggle.com/camnugent/sandp500

Pre-requisites:
- Python
  - pandas
  - numpy
  - matplotlib.pyplot
  - matplotlib.ticker
  - matplotlib.dates
  - datetime
  
Installing:
1. Download the dataset (.csv)
2. Download the functional program (.py)

Running:
1. Open the functional program (.py)
2. Within the Python Shell, enter "tickers()" to evaluate the symbols/tickers of the stocks within the dataset
3. Within the Python Shell, enter "AnnualReview()" to evaluate the data for a particular stock. 
    The application will prompt you for information including the stock's symbol (e.g., AAPL) as well as whether you would like to look at highs/lows.
4. View any outputs (.png images of data plots) in the folder where your functional program and dataset is saved
